Wi-Fi geolocation tool - это небольшая утилита для определения координат пользователя при помощи глобальных баз данных точек доступа Wi-Fi без использования GPS.

Wi-Fi geolocation tool поддерживает следующие способы хранения API-токена:
  * переменные окружения (токены будут сохранены в переменных окружения текущего пользователя системы) — используется по умолчанию;
  * скрипт запуска (токены будут сохранены непосредственно внутри автоматически сгенерированного скрипта) — устаревший, не рекомендуется;
  * ручной режим - токены не будет сохраняться и должны быть переданы вручную при помощи переменных окружения среды.

Персональные токены можно получить здесь:
  * https://developers.google.com/maps/documentation/geolocation/intro;
  * https://ichnaea.readthedocs.io/en/latest/api/index.html;
  * https://yandex.ru/dev/locator/doc/dg/api/geolocation-api.html.
